# Career Type Check

キャリアタイプ12診断ツール - 36問の質問であなたのキャリアタイプを診断します。

## 📋 概要

Career Type Checkは、36問の質問に答えることで、12のキャリアタイプからあなたに最も適したタイプを診断するWebアプリケーションです。5つの軸（突破・行動力、対人・影響力、論理・戦略力、創造・企画力、規律・完遂力）で特性を可視化し、具体的なキャリアキーワードを提示します。

**デモサイト**: [career-type-check.vercel.app](https://career-type-check.vercel.app)

## ✨ 主な機能

- **36問の診断質問**: 4択形式（あてはまる/ややあてはまる/あまりあてはまらない/あてはまらない）
- **12のキャリアタイプ**: 4つの群（A: 開拓・牽引、B: 調和・支援、C: 創造・戦略、D: 実務・品質）に分類
- **5軸レーダーチャート**: あなたの特性を視覚的に可視化
- **進捗保存機能**: LocalStorageを使用して回答の途中経過を保存
- **レスポンシブデザイン**: モバイル・デスクトップに対応
- **ミニマルUI**: 落ち着いた、信頼性の高いデザイン

## 🎯 12のキャリアタイプ

### A群：開拓・牽引（対人 × 革新）
1. **ザ・パイオニア** - 開拓者：行動力がMAX。細かい管理は苦手だが、道を切り開くパワーが突出。
2. **ザ・ドライバー** - 達成者：行動と論理が高いバランス。成果を出すための推進力が強い。
3. **ザ・インフルエンサー** - 発信者：対人と創造がMAX。ロジックよりも感性と人を惹きつける力が武器。

### B群：調和・支援（対人 × 安定）
4. **ザ・コネクター** - つなぐ人：対人力がMAXで他も平均以上。組織の潤滑油として欠点が少ない。
5. **ザ・サポーター** - 支援者：対人と規律がMAX。自分が前に出るより支えることに特化。
6. **ザ・エデュケーター** - 育成者：対人が高く、論理的に教えることもできる。全体的に安定したバランス。

### C群：創造・戦略（対事 × 革新）
7. **ザ・ヴィジョナリー** - 構想者：論理と創造がMAX。未来を描くことに特化し、実務は他者に任せる。
8. **ザ・クリエイター** - 創造者：創造性がMAX。常識に縛られないため規律は最低値だが、爆発力がある。
9. **ザ・ハッカー** - 改善者：論理と行動が高い。「効率化」のために既存ルールを壊すため規律は低め。

### D群：実務・品質（対事 × 安定）
10. **ザ・スペシャリスト** - 職人：論理と規律がMAX。対人は最低値だが、圧倒的な技術と品質を誇る。
11. **ザ・ガーディアン** - 守護者：規律がMAX。リスクを検知し、組織をミスから守る守備の要。
12. **ザ・アーキテクト** - 設計者：論理と規律が高い。感情に左右されず、淡々と仕組みを構築・運用する。

## 🚀 使い方

### ローカルで実行

1. リポジトリをクローン
```bash
git clone https://github.com/Aramaki70ya/career-type-check
cd career-type-check
```

2. ローカルサーバーで起動
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合（http-serverを使用）
npx http-server -p 8000
```

3. ブラウザでアクセス
```
http://localhost:8000/キャリアタイプチェック/app/
```

### Vercelでデプロイ

このプロジェクトはVercelにデプロイされています。`vercel.json`の設定により、`/キャリアタイプチェック/app/`がルートとして設定されています。

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: カスタムプロパティ、Flexbox、Grid
- **Vanilla JavaScript**: フレームワークなしの純粋なJavaScript
- **LocalStorage API**: 進捗保存機能
- **Chart.js** (推測): レーダーチャート描画

## 📊 診断の仕組み

1. **質問回答**: 36問の質問に4択で回答（0-3点）
2. **スコア計算**: 各タイプの質問（3問ずつ）の合計点を計算
3. **タイプ判定**: 最もスコアが高いタイプを診断結果として表示
4. **レーダーチャート**: 5軸（行動力、対人力、論理力、創造力、規律力）のスコアを可視化
5. **キーワード提示**: 診断されたタイプに適したキャリアキーワードを表示

## 🎨 デザイン原則

このプロジェクトは以下のデザイン原則に基づいています（`.agent/rules/design-rules.md`参照）：

- **ミニマルデザイン**: 落ち着いた、信頼できる、余白のあるデザイン
- **カラーパレット**: 白を基調とし、使用色は三色以内に限定
- **タイポグラフィ**: サンセリフを基本とし、太字の多用は禁止
- **レイアウト**: 余白を大胆に取り、GridよりもColumnレイアウトを優先
