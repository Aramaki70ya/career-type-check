## デザイン統一ルール

| 項目 | 設定 |
|------|------|
| 画風 | 絵本イラスト風、水彩テクスチャ、親しみやすい雰囲気 |
| 構図 | 全身〜バストアップ、正面〜やや斜め向き |
| 背景 | シンプルな水彩ウォッシュ（各キャラのテーマカラー） |
| 表情 | 各タイプの特性を表す（活発、穏やか、知的など） |
| スタイル | 擬人化動物（ケモノ）、二足歩行、程よくかわいい |
| テイスト | チビすぎず、リアルすぎず、絵本風のちょうど良いバランス |
