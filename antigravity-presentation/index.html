Why Antigravity?

なぜ今、これを激推しするのか

相談相手ではなく、
「実行部隊」として。

ChatGPTやGeminiとの対話は「相談」です。しかし、Antigravityは違います。
それは「実行部隊」です。

コードを読み、計画を立て、修正し、テストする。
この実行力こそが、私たちがAntigravityを選ぶ理由です。

Basic Operations

シンプルで強力な画面構成と操作

3つのエリアを使いこなす

  • 📁 左：ファイルエリア & メモ
    ここに情報を溜めることがコンテキストエンジニアリングの要。
  • 📝 中：エディタ
    コードやドキュメントを閲覧・編集する場所。
  • 🤖 右：エージェントチャット
    指示を出し、実行させる司令塔エリア。

情報を溜めて、参照させる

エージェントチャットでは以下の操作が可能です：

  • @ ファイルメンション: ドラッグ&ドロップでもOK。
  • / コマンド: Workflowsなどの機能を呼び出し。
  • 🖼️ 画像入力: スクショなどもそのまま貼れる。

💡 左側にメモを積み上げ、右側でそれを参照させながら指示を出す。
これがコンテキストエンジニアリングの基本です。

Powerful Features

非エンジニアでも使いこなせる機能群

Rule Settings

Global（全PJ共通）とWorkspace（PJ固有）の2種類のルールを設定可能。AIの振る舞いを細かく制御。

Planning Mode

AIが勝手に「Implementation Plan」を作成。Wordのようにコメントを入れるだけでフィードバック完了。

Walkthrough

AIによる作業報告書。実行したタスクをスクショや録画で確認。コメントだけで再実行依頼が可能。

Screenshot FB

Webサイトのスクショに直接書き込み。非エンジニアでも的確なUI修正指示が出せます。

Nano Banana

「UIイメージを提案して」と言うだけ。デザイナーのように画像を生成し、提案してくれます。
